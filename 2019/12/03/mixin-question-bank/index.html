mixin签到答题库 会不断更新

本身答题任务所出题目并不难，只要在Mixin生存一段时间的小伙伴都可以做对。所以，这份题库更多的是给刚刚进入Mixin的小白所写，希望看了我的题库，你可以轻轻松松学到知识，持续签到领取比特币。

001 在 Mixin 钱包里的钱什么情况下可能丢失？

A 我的电话号码丢失，那我就会丢失我的资产 B 项目方不再运营项目，那我的钱就无法访问了 C Mixin 的服务器被黑客攻击会导致用户资产丢失 D Mixin 钱包的电话和 6 位密码是唯一证明，只有同时拿到了电话和 6 位密码才能转移资产

答案：D

解析： Mixin是一个去中心化的资产存储网络，你的6位密码只有你自己知道。除了你，谁也转移不了你的资产。

002 签到领取的 BTC 在哪里？

A 项目方的积分里 B 自己的私人钱包里 C 自己在项目方的数据库积分 D 项目方的钱包里

答案：B 解析： 注册Mixin之后一定要牢记自己的6位数密码，正是因为Mixin是去中心化的钱包。所以，除了拥有6位数密码的你，谁也动不了你的资产。它就会乖乖的躺在你的私人钱包里。

003 我签到的 BTC 可以卖掉吗？

A 不可以卖掉，因为这只是一个数字 B 可以卖掉，因为签到的 BTC 已经在自己钱包里了 C 可以卖掉，但是需要项目方同意 D 不可以卖掉，因为这不是真的 BTC

答案：B 解析： 接第二题，既然签到获得的BTC在你的私人钱包里了，你可以卖掉它。

004 项目方能不能冻结我的 Mixin 钱包？

A Mixin 钱包是私人财产，法律要求 Mixin 不能冻结我的钱包 B Mixin 钱包是项目方帮我管理的，所以他们可以冻结 C Mixin 钱包是私人财产，道德要求 Mixin 不能冻结我的钱包 D Mixin 钱包是私人财产，技术上任何人都不能冻结

答案：D 解析： 去中心化的钱包，不存在中心服务器，不像银行一样可以冻结你的资产。所以，没有人能够冻结你的钱包。

005 端到端加密有什么好处？

A 聊天内容是加密的，只有聊天双方可以知道聊天内容，任何中间人包括 Mixin 项目方都不知道 B 聊天内容是加密的，中间人不能劫持聊天消息，但是 Mixin 项目方可以知道 C 聊天内容是加密的，中间人可以劫持聊天消息，但是 Mixin 项目方不知道 D 聊天内容是加密的，中间人可以劫持聊天消息，Mixin 项目方也知道

答案：A 解析： Mixin作为加密聊天工具，采用了严格的端对端加密技术，信息在双方手机发出时即完成加密，服务器处理的信息是经过加密不可被破译的，完全保证了社交信息的安全。

006 普通人的 BTC 放在哪里最安全？

A 放在冷钱包里最安全 B 换成人民币放在支付宝里最安全 C 放在 Mixin 钱包里已经是最安全最方便的了 D 放在交易所里最安全

答案：C 解析： 这个问题参照Mixin创始人冯晓东的一篇博文：什么是最安全的数字资产管理方案

007 如果不是有效用户，下面哪个说法是正确的？

A 可以继续签到，只要每天坚持就好 B 有效用户的说法只是项目方故意为难用户 C 只要签到就是有效用户 D 当系统检查到不是有效用户时会中断甚至永久终止相关用户签到

答案：D 解析： Mixin肯定会优先保障有效用户的权益，当发现你只是每天来签到，不用Mixin聊天，不存储数字货币在Mixin上，不用它来交易和转账时，Mixin有权利中断甚至终止你的签到活动。

008 我签到的 BTC 项目方可以收回吗？

A 不可以收回，因为这不是真的 BTC，项目方没有收回的必要 B 不可以收回，因为签到的 BTC 已经在自己钱包里，项目方如果转移是违法的 C 不可以收回，因为签到的 BTC 已经在自己钱包里，技术上只有我自己可以转移 D 可以收回，因为签到的 BTC 只是一个数字，项目方随时可以更改

答案：C 解析： 只有拥有6位数密码的人才可以转移资产，甚至如果你自己丢失了6位数密码，你的资产也就找不回来了。

009 签到中间出现各种任务的目的是什么？

A 通过复杂任务，让用户断签 B 过滤无效用户，保障真实有效用户的权益 C 故意为难签到用户，让用户无法继续签到 D 项目方没钱了

答案：B 解析： 跟007题连起来一起看，签到任务是为了剔除羊毛党和无效用户，把比特币发给真正用Mixin的用户。

010 Mixin 钱包是去中心化的吗？

A 不是，Mixin 是跟中心化交易所类似的 B 不是，Mixin 是项目方中心化管理的钱包 C 是的，但是 Mixin 的钱包不安全 D 是的，Mixin 是去中心化钱包里普通用户使用最安全最方便的

答案：D 解析： 参见006题答案解析

011 哪个行为不是 Mixin 的有效用户？

A 把自己的加密货币存放在 Mixin B 每天坚持点一下签到领取 BTC C 经常在 Mixin 交易和转账 D 在 Mixin 跟朋友安全的聊天

答案：B 解析： 每天只是登录一下来签到的是羊毛党，有效用户要么把Mixin当做聊天软件用，要么把它当做多币种钱包，要么用它来交易和转账数字货币。

012 下面哪个对于 Mixin 钱包的描述是不准确的？

A 相比较其他钱包，Mixin 具有明显的安全性和易用性 B 钱包是中心化的 C 在技术实力、管理能力和监管透明度上都是最优秀的，开源的，去中心化的钱包 D 团队强大，拥有稳定管理数字资产多年的能力，具有很高的知名度

答案：B 解析： Mixin钱包是去中心化钱包。

013 强制端到端加密的聊天工具是哪个？

A Mixin B 微信 C QQ D Telegram

答案：A 解析： Mixin是强制的端对端的加密聊天工具，参见005题解析。

014 签到领取的 BTC 为什么有时候到账很慢？

A 因为项目方没钱了故意不转账 B 因为 Mixin 网络转账很慢 C 因为签到的人非常多，机器人需要按照顺序转账，可能需要几个小时 D 因为比特币网络转账很慢

答案：C 解析： 每天会有十几万个人在Mixin签到，机器人要挨着发放奖励。大多数人又会在早上8点多一起签到，这会令机器人不堪重负。

015 下面哪个不是 Mixin 的核心功能？

A 签到领取 BTC B 创建各种功能的机器人 C 端到端加密聊天 D 最安全方便的 BTC 钱包

答案：A 解析： Mixin是一个集合聊天+全币种钱包+易用开发平台为一体的工具。签到只是一种吸引用户，让其应用Mixin的手段。

016 下面哪一个功能 Mixin 还没有？

A 发送语音消息 B 转账人民币 C 语音电话 D 看视频直播

答案：B 解析： Mixin是一个全币种的数字资产钱包，可以发语言，看视频直播，打语音电话。但是，不能转账人民币。

017 Mixin 的 6 位数字密码忘记怎么办？

A 重新设置一个密码 B 密码不能忘记，一旦忘记资产永久丢失 C 联系客服找回密码 D 通过短信验证找回密码

答案：B 解析： 去中心化钱包，就是要自己保存密码，一旦丢失资产就再也找不回来了。
