手机验证教程：

最近更新出来了手机认证，完成手机认证具体有什么用还不知道，但是对于判定你是不是机器人，还是对于忘记密码后找回密码有莫大的帮助，既然出来了这个东西，肯定是有用的，今天教大家花几分钟完成他，免得白挖币。

这个界面最下面那个按钮千万不要点，是退出APP的，退出可能币就消失了
